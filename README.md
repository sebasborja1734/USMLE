# USMLE Tracker

A web application for tracking progress on the USMLE exams.

## Features
- Track study schedules
- Monitor progress
- Organize study materials

## Installation

To run this project:
1. Clone this repo.
2. Navigate to the directory and run `npm install`.
3. Start the development server with `npm run dev`.
